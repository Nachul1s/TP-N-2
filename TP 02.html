¿qué es una URL?

Una URL (Uniform Resource Locator) es un enlace subdividido en diferentes segmentos como protocolo, nombre del host, nombre del sitio, dominio, path del directorio, path (ruta) absoluta y el documento, entre otros

¿qué es una url absoluta?

¿qué es una url relativa?

Las URL relativas describen la ruta a un archivo relativo al actual documento

¿cuál es la diferencia entre una url relativa y una url absoluta?

La URL relativa es la parte donde termina el dominio y empieza el path del directorio y lo absoluto es del protocolo hasta el dominio

¿Cuál es el propósito de la etiqueta anchor?

El proposito de la etiqueta anchor insertar una URL y entre los signos de mayor y menor, ponerle el nombre a esta url

¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

El atributo que se utiliza es "href"

¿Qué sucede si el atributo "href" en la etiqueta anchor está vacío?

No mostrara nada

¿Cuál es el uso del atributo "target" en la etiqueta anchor y cuáles son los posibles valores que puede tener?

El uso del target es que es un archivo que contiene elementos, propiedades , destinos y tareas para escenarios comunes.
